# 课程资料链接查询插件

## 简介
此 Python 程序是一个课程资料链接查询插件，它能够依据用户输入的课程名称，在指定的 CSV 文件里查找匹配的课程记录，并且从 MinIO 对象存储服务中获取该课程资料的预签名下载链接。若未找到匹配的课程记录或者资料文件，程序会返回相应的错误提示信息。

## 功能特性
- 接收用户输入的课程名称。
- 检查 CSV 文件是否包含必需的字段。
- 在 CSV 文件中精确匹配课程名称。
- 从 MinIO 生成课程资料的预签名下载链接。
- 处理文件未找到、MinIO 错误以及系统错误等异常情况。
